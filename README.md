## installing/running

### python install
1. clone the repo
2. install the requirements.txt file
3. use the .env.template file as a guide for making your .env file
4. now run main.py

### C install
#### Option 1
1. compile main.c from releases

#### Option 2
1. download a precompiled version for your system
